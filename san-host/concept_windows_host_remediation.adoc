= Windows主機補救
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
如果您使用7-Mode Transition Tool（7MTT）從Data ONTAP 以7-Mode運作的VMware移轉至叢集Data ONTAP 式VMware、則必須執行特定步驟、準備Windows主機進行移轉。您也必須執行特定步驟來準備轉換階段、而且在轉換之後、您必須讓Windows主機上線。
